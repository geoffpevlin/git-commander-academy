How to Play Git Commander Academy

The definition of a git command will appear on the screen.

Using your keyboard or the on-screen keyboard, type the answer to the git command definition.

Each empty box represents one letter in the correct response. Press enter when you are ready to submit your answer.

If you are correct, you will receive the dollar amount displayed and will move to the next definition. If you are incorrect, the letters will turn red. You backspace to delete your letters and try again.

If you do not know the answer, click Skip! But you will not git a second chance on any skipped definition!
